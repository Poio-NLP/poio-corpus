Poio

Technologies for Language Diversity

Poio is a free and open source predictive text and transliteration system for lesser-used languages. We support text input for languages like Bavarian, Chechen and Sorbian on mobile phones and desktop computers

The Poio Corpus is a freely available collection of language resources for the lesser-used languages. The data is extracted from free sources like Wikipedia, dictionaries, documents, websites and others. Each corpus is available for download as GrAF files (ISO 24612) for which a Python and a Java library are available to allow unified access. The goal of this project is to make the development of NLP tools for the lesser-used languages easier, or even possible in some cases. Our website features two example applications for all languages: lexical graphs to visualize semantic similarities based on the idea of Widdows and Dorow (2002) and an text editor with statistical text prediction. All the functionality of this website is also available via a Web API.

The Poio Corpus is a project of the Interdisciplinary Centre for Social and Language Documentation which is dedicated to the documentation and preservation of the linguistic and cultural heritage in Europe.
